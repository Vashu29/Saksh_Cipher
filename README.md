🚀 Project Showcase: Saksh Cipher (Android Application)
I’m excited to share one of my academic projects that I developed during my 5th semester (Oct 2024 – Nov 2024) but posting about it now.
🔐 Saksh Cipher is an Android application built with a focus on classical cryptography.
 Key features:
Implemented multiple encryption & decryption algorithms (Vigenère, Playfair, Caesar, Hill, Vernam [2*2] as well as [3*3], etc.).
Provides a secure notepad and an SMS module for encrypted communication.
Designed for 100% efficiency in text security operations.
💡 Working on this project significantly enhanced my logical thinking, ability to write clean code, and solve complex problems—skills that are essential for a software engineer.
This project also deepened my understanding of cryptography techniques by not just implementing algorithms but also learning how they function under the hood.
Grateful for this learning experience and looking forward to applying these problem-solving skills in larger, real-world projects! 🌐✨
